# jackal
Jackal board game


#### Author of code
Vladislav Kuleikin


## How to play
1. Install all required packaged from `requirements.txt`.
2. Run next command in cmd:
```cmd
python App.py
```

### Controls
* Mouse clicks - for the movement
* Enter - pick up/down objects
* Alt - change character
* R - start new game
* Esc - exit the game
